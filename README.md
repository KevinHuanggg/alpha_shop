# Alpha Shop

使用 React 搭建電商平台的購物車結帳畫面。

## 功能

- 顯示當前結帳流程進度
- 人員付款資訊填寫欄位
- 結帳流程進退控制按鈕

## 安裝與執行

1. 請先確認環境有安裝 node.js 與 npm
2. 將專案 clone 到本地
3. 在本地開啟後，透過終端機進入資料夾，輸入:

```
npm install
```

4. 安裝完畢後，繼續輸入:

```
npm start
```

5. 打開瀏覽器進入到此網址: http://localhost:3000
6. 若欲暫停使用， 輸入:

```
ctrl + c
```

## 開發工具

- Node.js 18.16.0
- React 18.2.0
